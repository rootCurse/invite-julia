## Стиль

- **Палитра:** чёрный + белый. Минималистичный монохром, высокий контраст, строгая типографика (без золота/бордо).
- **Референсы:** не заданы — подбор на усмотрение дизайнера, опираться на раздел 3 `invite.md`.
- **Лого/монограмма:** нет готового — создаётся с нуля (монограмма «А & Ю»).
- **Фото:** нет — использовать плейсхолдеры (для обложки, love story, галереи).
- **Шрифты:** пара **Yeseva One** (заголовки/даты) + **Montserrat** (текст, UI, кнопки, формы) — обе с поддержкой кириллицы, чёткий контраст засечка/гротеск подходит монохромной строгой эстетике. Для монограммы/имён на обложке — акцент через **Marck Script** только в декоративном элементе (не для длинного текста).

## Контент

- **Love story:** блок нужен, текст — Lorem ipsum как плейсхолдер.
- **Доп. блоки, которые включаем:** История пары (love story), анкета для гостей (вопросы — заглушки, уточнят позже), блок «Второй день свадьбы» (программа — заглушка).
- **Контакты организатора:** заглушка (имя, телефон/Telegram — placeholder).
- **Вишлист/реквизиты:** не нужен.
- **Текст приветствия к гостям:** авторский текст, придуман для макета (см. `content-structure.md`).
- **Именные ссылки / персональные обращения:** не нужны.

## RSVP и функциональность

- RSVP-форма нужна. Поля будут уточнены позже — для макета/MVP используется стандартный набор (имя, участие да/нет, кол-во гостей, дети, аллергии/пожелания по меню) — см. `content-structure.md`.
- Ответы гостей → **Google Таблица**.
- Адаптив: **полный** — мобильная версия и desktop одинаково важны.
- Многоязычность: не нужна.
